# Sternmeister Dialog Research

Мультиагентная система для исследования и анализа диалогов менеджеров по продажам компании Sternmeister.

## Описание

Система автоматически получает данные из CRM Kommo, транскрибирует записи звонков, проводит глубокий анализ по методологиям SPIN/BANT/Sandler и строит корреляции между речевыми паттернами менеджеров и CRM-статусами сделок. Результаты отображаются в интерактивном веб-дашборде.

## Возможности

### 📞 Боевой скрипт v5.1 (для менеджеров) — НОВОЕ

Главная вкладка для ежедневной работы менеджеров. Разговорная редакция скрипта GosBuh v5.1 в формате интерактивного справочника:

- **19 этапов** скрипта (от Pre-call ritual до Frame Control) с раскрывающимися блоками, хронометражем и обоснованием каждой фразы
- **22 возражения** с разговорными ответами и Voss-протоколом отработки
- **12 сегментов клиентов** (CS1.A главбух · CS2 Pflege · CS3 старт с нуля · CS4 декрет · ETHICAL 60+ и др.)
- **6 историй** выпускников с именами и точными цифрами (Елена · Анна · Светлана · Ирина · Махмеджон+Милия · Сергей)
- **WhatsApp** — 7-шаговый каденс касаний + 3 готовых шаблона
- **Чек-лист** mock-call evaluation с интерактивным подсчётом баллов
- **Шпаргалка** — must say / must not / Voss-протокол на одной странице
- **Поиск** по всему скрипту, **кнопки копирования** для каждой фразы

### Аналитика

- **CRM-интеграция** — автоматическое получение лидов, статусов и записей звонков из Kommo
- **Транскрипция** — распознавание речи через AssemblyAI с разделением спикеров
- **Глубокий анализ** — оценка каждого звонка по методологиям SPIN, BANT, Sandler через xAI Grok
- **Корреляционный анализ** — связь конкретных фраз менеджера с негативными CRM-исходами
- **Рекомендации** — data-driven корректировки скрипта продаж
- **Аудит скрипта** — сравнение текущего скрипта с реальной проблематикой 30 звонков, выявление 9 системных разрывов
- **Улучшенный скрипт v3.1** — оптимизирован для 20-25 мин консультации. 5 фаз с хронометражем: Захват → Квалификация → Переворот → Мост → Закрытие. 7 методологий (SPIN, Sandler, Chris Voss, Challenger, Gap Selling, Straight Line, CARE). 5 листов: основной скрипт, истории, возражения, WhatsApp-шаблоны, тональность
- **Excel-скрипт** — готовый файл `Sternmeister_Скрипт_v3.1.xlsx` для работы менеджеров
- **Веб-дашборд** — интерактивная визуализация с 8 страницами: обзор, корреляции, звонки, возражения, рекомендации, отчёт, аудит скрипта, новый скрипт
- **Анализ закрытых сделок** — 14 звонков из статуса «Закрыто и не реализовано» с категориями: график, семья/здоровье, цена, неактуально

## Архитектура (6 агентов)

| Агент | Файл | Задача |
|-------|------|--------|
| CRM Agent | `agents/fetch-crm-data.js` | Получение лидов и звонков из Kommo |
| Transcribe Agent | `agents/transcribe-calls.js` | Транскрипция через AssemblyAI |
| Analysis Agent | `agents/analyze-dialogs.js` | Анализ по SPIN/BANT/Sandler |
| Correlation Agent | `agents/build-correlations.js` | Матрица корреляций |
| Report Agent | `agents/generate-report.js` | Executive summary |
| Orchestrator | `agents/orchestrator.js` | Координация pipeline |

## Установка и запуск

```bash
npm install
```

Заполните `.env` ключами API (KOMMO, AssemblyAI, xAI).

```bash
# Запуск всего pipeline
npm run pipeline

# Или по шагам
npm run fetch        # Получить данные из CRM
npm run transcribe   # Транскрибировать звонки
npm run analyze      # Анализ через Grok
npm run correlate    # Построить корреляции
npm run report       # Сформировать отчёт

# Открыть дашборд
npm start
```
